# Task Manager

A React-based Task Manager application developed using React Hooks.

## Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Display completed and pending tasks separately
* Show total task count
* Show completed task count
* Show pending task count
* Local Storage support
* Responsive user interface

## Technologies Used

* React
* Vite
* JavaScript
* CSS

## React Hooks Used

### useState

Used for managing:

* Tasks list
* Input field value

### useEffect

Used for:

* Saving tasks to Local Storage
* Loading tasks from Local Storage when the application starts

## Installation

1. Open terminal

2. Install dependencies

npm install

3. Run the project

npm run dev

4. Open browser

http://localhost:5173
